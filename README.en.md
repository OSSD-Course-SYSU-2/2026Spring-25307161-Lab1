# SmartStudy Schedule
✨ A HarmonyOS ArkTS To-Do & Focus App Designed for College Students

## 📌 Introduction
This project is a secondary development based on Huawei's official HarmonyOS Codelab To-Do List example. It is deeply enhanced for college students' study and life scenarios, with core functions such as **multi-category to-do management, study points check-in, focus timer, and lock screen mode**, helping students better plan daily, study and review tasks.

## 🎯 Core Features
### 1. Multi-Category To-Do Management
- **Daily**: Campus life tasks such as morning running, volunteer activities, lectures, and picking up packages
- **Practice**: Professional course exercises such as Advanced Mathematics, Linear Algebra, University Physics, and Circuit Analysis
- **Vocabulary**: CET-4/CET-6 high-frequency vocabulary memorization
- **Review**: Knowledge consolidation for various courses
- **All**: Show all tasks across categories

### 2. Study Points System
- +5 points for completing daily tasks
- +10 points for completing study tasks
- +20 points for the first check-in of the day
- Persistent data storage


## 🛠 Tech Stack
- Language: ArkTS (HarmonyOS declarative syntax)
- IDE: DevEco Studio
- System: HarmonyOS 5.0+
- State Management: @State, @StorageProp
- Routing: @ohos.router

## 📱 How to Run
1. Install DevEco Studio 6.0+
2. Import the project
3. Connect a HarmonyOS device or emulator
4. Build and run directly
