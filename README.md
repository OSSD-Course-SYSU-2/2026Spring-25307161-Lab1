# 智学日程 (SmartStudy Schedule)
✨ 专为大学生定制的鸿蒙HarmonyOS学习待办&专注管理APP

## 📌 项目简介
本项目基于华为官方 Codelab 待办列表示例进行二次开发，面向大学生学习生活场景，深度增强了**多分类待办、积分打卡、专注计时、锁机模式**等核心功能，帮助学生更好地规划日常、学习与复习任务。

## 🎯 核心功能
### 1. 多分类待办管理
- 日常：晨跑、讲座、取快递等大学生生活任务
- 刷题：高数、线代、大物、电路等专业课练习
- 单词：四六级高频词汇背诵
- 复习：各科知识点巩固
- 全部：展示所有分类任务

### 2. 学习积分系统
- 完成日常任务 +5 积分
- 完成学习类任务 +10 积分
- 每日首次打卡 +20 积分
- 数据持久化存储


## 🛠 技术栈
- 开发语言：ArkTS（HarmonyOS 声明式语法）
- 开发工具：DevEco Studio
- 系统版本：HarmonyOS 5.0+
- 状态管理：@State、@StorageProp
- 页面路由：@ohos.router

## 📱 运行方式
1. 安装 DevEco Studio 6.0+
2. 导入项目
3. 连接 HarmonyOS 设备或模拟器
4. 直接运行 build & run
